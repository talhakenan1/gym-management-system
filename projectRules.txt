1. Gereksinim Analizi
Projenin temeli, ASP.NET Core MVC kullanılarak bir Spor Salonu Yönetim ve Randevu Sistemi geliştirmektir.

A. Teknik Gereksinimler (Kullanılacak Teknolojiler)
Platform: ASP.NET Core MVC (Güncel LTS sürümü) 
Programlama Dili: C# 
Veritabanı: SQL Server veya PostgreSQL 
ORM: Entity Framework Core
Sorgulama Dili: LINQ 
Front-End: Bootstrap 5 , HTML5, CSS3, JavaScript, jQuery 
Yapay Zeka: OpenAI API veya hazır bir model 

B. Fonksiyonel Gereksinimler (Sistemin Yapması Gerekenler)
1. Temel Sistem Özellikleri:
Arayüz (Front-End): Uygulamanın bir kullanıcı arayüzü olmalıdır.
Admin Paneli: Bir yönetici paneli bulunmalıdır.
Veri Doğrulama: Hem istemci (client) hem de sunucu (server) tarafında veri doğrulaması (data validation) yapılmalıdır.
2. Kimlik Doğrulama ve Yetkilendirme (Auth):
Kayıt/Üye Olma: Kullanıcı kayıt sayfası bulunmalıdır.
Rol Bazlı Yetkilendirme: Sistemde en az iki rol olmalıdır.
Admin: ogrencinumarasi@sakarya.edu.tr / Şifre: sau 
Üye: Kayıtlı kullanıcı 
Authorization (Yetkilendirme) kuralları doğru bir şekilde uygulanmalıdır.
3. Varlık Yönetimi (CRUD İşlemleri):
Tüm temel varlıklar (Spor Salonu, Antrenör, Hizmet, Üye, Randevu vb.) için CRUD (Create, Read, Update, Delete) işlemleri tam olarak uygulanmalıdır.
4. Modüller ve İşlevler:
Spor Salonu Tanımlamaları:
Sistemde spor salonları tanımlanabilmelidir (tek salonlu uygulama da kabul edilir).
Her salon için çalışma saatleri, hizmet türleri (fitness, yoga vb.), hizmet süreleri ve ücretleri tanımlanabilmelidir.
Antrenör (Eğitmen) Yönetimi:
Salonlarda görev yapan antrenörler sisteme eklenebilmelidir.
Antrenörlerin uzmanlık alanları (kas geliştirme, kilo verme vb.) ve verebildikleri hizmet türleri belirtilmelidir.
Antrenörlerin müsaitlik saatleri tanımlanabilmelidir.
Üye ve Randevu Sistemi:
Üyeler, uygun antrenör ve hizmete göre randevu alabilmelidir.
Randevu saati çakışıyorsa (uygun değilse) sistem kullanıcıyı uyarmalıdır.
Randevu detayları (hizmet, süre, ücret, eğitmen) sistemde saklanmalıdır.
Randevu onay mekanizması bulunmalıdır.
REST API Kullanımı:
Projenin en az bir bölümü, veritabanı ile REST API kullanarak iletişim kurmalıdır.
API üzerinden LINQ sorguları ile filtreleme (en az bir modülde) yapılmalıdır.
Örnek API fonksiyonları: Tüm antrenörleri listeleme , belirli tarihte uygun antrenörleri getirme , üye randevularını getirme.

Yapay Zeka Entegrasyonu:
Yapay zeka aracı ile entegre bir özellik bulunmalıdır.
Kullanıcılar fotoğraf yükleyerek veya vücut tipi/boy/kilo bilgisi girerek egzersiz veya diyet planı önerileri alabilmelidir.
(İsteğe bağlı): Kullanıcıların belirli egzersizleri yapınca nasıl görüneceklerine dair bir resim istenmesi.
